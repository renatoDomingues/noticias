## Install:
````
* yarn add express ejs
* yarn add mongoose
* yarn add bcrypt
* yarn add express-session
* yarn add body-parser
* yarn add passport
* yarn add passport-local
* yarn add passport-facebook
* yarn add passport-google-oauth
````

## Adicionar package:
````
"start": "node index.js"
````

## Terminal
````
* node index.js
* npm start
````
